# SQL Business Analysis

## Overview

This project contains SQL solutions for real-world business scenarios using a movie rental database.

The objective is to demonstrate SQL skills by solving analytical business questions related to marketing, finance, human resources, and store performance.

## Skills Demonstrated

- SQL Joins
- Common Table Expressions (CTE)
- Subqueries
- Aggregation Functions
- CASE Statements
- Filtering and Sorting
- Business Data Analysis

## Business Questions Solved

### Marketing Analysis
- Animation movies released between 2017–2019
- Top 5 actors by number of movies since 2015
- Genre production trends (Drama, Travel, Documentary)
- Most rented movies and target audience age groups

### Financial Analysis
- Revenue generated by each store after March 2017
- Store profitability analysis

### Human Resources Analysis
- Top 3 employees by revenue generated in 2017

### Actor Career Analysis
- Actor inactivity periods
- Gaps between movie releases

## Database

The queries were written for the PostgreSQL DVD Rental sample database.
